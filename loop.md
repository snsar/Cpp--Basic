### Bài 1: In dãy số chẵn lẻ

**Đề bài:**
Nhập số nguyên `n` (≥ 1). In ra các số từ 1 đến `n`, mỗi số cách nhau một dấu cách. Nếu số là chẵn in thêm `(EVEN)` sau số, nếu là lẻ in `(ODD)`.

**Input:**

```
n
```

**Output:**
Dãy số từ 1 đến `n` theo định dạng `<số>(EVEN)` hoặc `<số>(ODD)`.

**Ví dụ:**
Input:

```
5
```

Output:

```
1(ODD) 2(EVEN) 3(ODD) 4(EVEN) 5(ODD)
```

###  Bài 2: Tính tổng cho đến khi gặp số âm

**Đề bài:**
Nhập dãy số nguyên. Khi nhập số âm đầu tiên thì dừng lại. Tính tổng tất cả các số không âm đã nhập.

**Input:**
Một dãy số nguyên, kết thúc khi nhập số âm.

**Output:**
Tổng các số không âm.

**Ví dụ:**
Input:

```
4 7 0 3 -5
```

Output:

```
14
```

###  Bài 3: In tam giác sao ngược

**Đề bài:**
Nhập số nguyên `n`. In ra tam giác sao ngược gồm `n` dòng.
Dòng đầu có `n` dấu `*`, dòng sau ít hơn 1 dấu.

**Input:**

```
n
```

**Output:**
Tam giác sao ngược.

**Ví dụ:**
Input:

```
4
```

Output:

```
****
***
**
*
```

###  Bài 4: Đếm số chia hết cho 3 trong đoạn

**Đề bài:**
Nhập hai số nguyên `a` và `b` (a ≤ b). Đếm có bao nhiêu số trong đoạn `[a, b]` chia hết cho 3.

**Input:**

```
a b
```

**Output:**
Số lượng số chia hết cho 3 trong đoạn.

**Ví dụ:**
Input:

```
3 10
```

Output:

```
3
```

---

###  Bài 5: Tìm 5 số nguyên tố đầu tiên ≤ N

**Đề bài:**
Nhập số nguyên `N` (≥ 2). In ra tối đa 5 số nguyên tố nhỏ hơn hoặc bằng `N`. Nếu có ít hơn 5 thì in tất cả.

**Input:**

```
N
```

**Output:**
Các số nguyên tố nhỏ hơn hoặc bằng `N`, cách nhau bởi dấu cách.

**Ví dụ:**
Input:

```
20
```

Output:

```
2 3 5 7 11
```

---

###  Bài 6: Đảo ngược số và đếm chữ số

**Đề bài:**
Nhập số nguyên dương `X`. In ra số lượng chữ số và số đảo ngược của `X`.

**Input:**

```
X
```

**Output:**
Hai dòng:

* `Number of digits = D`
* `Reversed = Y`

**Ví dụ:**
Input:

```
12345
```

Output:

```
Number of digits = 5
Reversed = 54321
```

###  Bài 7: Bỏ qua số chia hết cho 5

**Đề bài:**
Nhập số nguyên `n`. In ra các số từ 1 đến `n`, **bỏ qua** các số chia hết cho 5 (dùng `continue`).

**Input:**

```
n
```

**Output:**
Các số từ 1 đến `n` trừ các số chia hết cho 5.

**Ví dụ:**
Input:

```
12
```

Output:

```
1 2 3 4 6 7 8 9 11 12
```
